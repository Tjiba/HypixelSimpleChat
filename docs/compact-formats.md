# Formats compacts — à remplir à la main

Remplis la ligne `COMPACT:` avec le rendu voulu, en codes `§`.
Laisse vide si tu préfères **GREY** (grisé), écris `HIDE` pour masquer, ou `OFF` pour garder tel quel.

**Codes couleur :** `§0`noir `§1`bleu foncé `§2`vert foncé `§3`cyan `§4`rouge foncé `§5`violet `§6`or `§7`gris `§8`gris foncé `§9`bleu `§a`vert `§b`aqua `§c`rouge `§d`rose `§e`jaune `§f`blanc
Exemple : `§6Hyperion §7· §c2.5M`
