## 👀 30 seconds of Hypixel chat, fixed

| Hypixel sends | You see |
|---|---|
| `Your Hyperion hit 3 enemies for 2,500,000 damage.` | `Hyperion · 2.5M` |
| `RARE REWARD! Player found a Fuming Potato Book in their Bedrock Chest!` | `RARE REWARD Player · Fuming Potato Book in Bedrock Chest` |
| `You earned 211 GEXP + 633 Event EXP from playing SkyBlock!` | `211 GEXP + 633 Event EXP` |
| `Guild > [MVP+] SomeBot: [V2] DiscordUser: gg` | `V2 DiscordUser: gg` |
| `[Sacks] +64 Cobblestone` | `+64 Cobblestone` |
| *That message repeated 14 times* | *One line ·* `(x14)` |

Same info. A fraction of the clutter. **Nothing deleted behind your back** — a beautifier, not a censor.

## ⚡ The big three

**🎛️ You decide, per message.** Every message type gets a dropdown — keep, dim, compact or hide — with a **live preview** while you tweak. 60+ rules across Lobby, SkyBlock, Dungeons, Bazaar, Slayer and more.

**🗂️ A chat you can actually use.** Tabs (All / Party / Guild), **Ctrl+F live search**, right-click any line to copy it, and up to **2048 messages** of history instead of 100.

**🎨 One consistent look.** Pick a theme color and every compact message follows it. Rarity colors on drops stay untouched.

<details>
<summary><b>📋 Full feature list</b></summary>

- Per-channel formatting (Public / Party / Guild / Officer) — custom prefixes, colors, ranks
- Guild **Discord bridge** cleanup with V1/V2/V3 tags
- Auto-collapse repeated messages with a `(xN)` counter — smart mode merges lines differing only by numbers
- Chat tabs: route what you send (`/pc`, `/gc`) or filter what you see
- Ctrl+F search, right-click copy, extended history
- Custom hidden patterns (your own blacklist)
- Optional `[HH:MM]` timestamps
- Modern config menu — settings search, tooltips, HSV color picker, one-click presets
- Master switch: off = chat 100% vanilla
- Client-side only · ~320 KB · **zero bundled dependencies** · update checker

</details>

## 📦 Install

Fabric Loader + [Fabric API](https://modrinth.com/mod/fabric-api) + [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin), drop the jar in `mods`, run `/hsc`. MC **26.1.2** and **26.2**.

Need the bridge itself? → **[Minecraft-Bridge-Chat](https://github.com/FrenchLegacy/Minecraft-Bridge-Chat)**
