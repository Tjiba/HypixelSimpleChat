## ✅ What you get

**Cleanup**
- ✂️ **60+ compact formats** — damage, sacks, rewards, dungeons, bazaar, slayer, boosters, NPC…
- 🎚️ Per-message control: `KEEP` · `DIM` · `COMPACT` · `HIDE` — with **live preview**
- 🔁 Auto-collapse repeats into `(xN)` — smart mode merges lines differing only by numbers
- 🚫 Custom blacklist — hide any text you type in
- 🧵 Guild **Discord bridge** cleanup — bot alias, real username, `V1/V2/V3` tags

**Chat superpowers**
- 🗂️ **Tabs** — All / Party / Guild : route your messages or filter the view
- 🔎 **Ctrl+F** — live search in chat
- 📋 **Right-click = copy** any line
- 🕓 **2048 messages** history (vanilla: 100)
- ⏱️ Optional `[HH:MM]` timestamps

**Style**
- 🎨 **Theme color** — every compact message follows your color, rarity colors preserved
- 🖌️ Per-channel prefixes, colors and rank toggles (Public / Party / Guild / Officer)
- 🖥️ Modern config menu — settings **search**, tooltips, HSV color picker, one-click **presets**

**Peace of mind**
- 🔌 Master switch → chat 100% vanilla
- 🪶 ~320 KB, **zero bundled dependencies**, client-side only
- 🔓 Open source (LGPL-3.0) · update checker · MC **26.1.2 & 26.2**

## 📦 Install

Fabric Loader + [Fabric API](https://modrinth.com/mod/fabric-api) + [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin) → jar in `mods` → `/hsc`
